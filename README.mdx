# Dynamic Stylings

Small Project to test out some fancy CSS styles.

## Current Styles

- dynamic dot grid background (at the position of the cursor, the dots get highlighted in a radius)
![Dynamic Dotgrid Demo](./assets/dynamic-dotgrid-demo.gif)
- dynamic cursor (glowing dot as normal cursor, pulses when hovering of interaction elements, transforms to a text cursor when hovering over text elements)
![Dynamic Cursor Demo](./assets/dynamic-cursor-demo.gif)
